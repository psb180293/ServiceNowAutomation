Github Link: https://github.com/psb180293/ServiceNowAutomation.git

Requirements: 

1) RAM : Minimum of 2GB , preferably 4 GB 
2) Java 1.8 + 
3) IDE to import the code(Eclipse/IntelliJ) (Optional)
4) Selenium JARs(provided in the directory)
5) Chromedriver.exe file

Prerequisite : 

1) Get the instance link
2) Get the user name
3) Get the password

How to run : 

1) Import the project to the IDE of your choice
2) Go the path /src/servicenow/
3) Open the file "RunMe.java"
4) provide your instance name,user name,password and the path of chromedriver.exe file
5) Run the java file
